Student Performance Analytics (SQL & Excel)

This project analyzes student performance using SQL, Excel, and basic data visualization.
It includes database queries, student score calculations, grade reports, and insights into academic performance.
Project Overview

The goal of this project is to:
	•	Clean and organize student score data
	•	Calculate total, average, and grade
	•	Find top-performing students
	•	Analyze subject-level and gender-level performance
	•	Visualize grade distribution
Tools Used
	•	SQL (Data analysis & querying)
	•	Excel (Data cleaning & report creation)
	•	Power BI (optional extension if you want a dashboard)
Key SQL Queries

Some insights generated using SQL:
	•	Calculate average score
	•	Find highest scoring student
	•	Subject-wise maximum marks
	•	Gender-wise performance
	•	Section toppers
	•	Ranking students based on averages
